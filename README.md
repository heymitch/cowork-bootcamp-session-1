# Cowork Bootcamp — Session 1: Setup & Organization

Your first session. Set up your agent, teach it who you are, and watch it organize your computer.

## Install

Add the marketplace, then install the plugin:

```
/plugin marketplace add heymitch/cowork-bootcamp-session-1
/plugin install cowork-bootcamp-session-1@cowork-bootcamp-session-1
```

One install — all three skills appear automatically.

## Skills

| Skill | Trigger | What It Does |
|-------|---------|-------------|
| System Prompt Settings | Read the skill file | Paste a system prompt so Claude knows how to work with you |
| Business Blueprint | "Run my business blueprint" | 5-min interview that creates config.md and CLAUDE.md |
| Organize Machine | "Organize my machine" | Sort your Desktop and Downloads using GTD principles |

## Session Order

1. **System Prompt Settings** — paste the prompt into Claude settings (30 seconds)
2. **Business Blueprint** — answer questions so your agent knows your business (5 minutes)
3. **Organize Machine** — watch your agent clean up your files (10-20 minutes)
